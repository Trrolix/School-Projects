# Szkolne Ćwiczenia HTML/CSS

To repozytorium zawiera zestaw prostych ćwiczeń HTML i CSS wykonanych podczas zajęć z projektowania stron internetowych w technikum.  
Niektóre pliki opierają się na materiałach edukacyjnych od nauczyciela, a część zawiera moje własne modyfikacje i ulepszenia.

## Co znajduje się w środku?

- Podstawowa struktura HTML  
- Listy i tabele  
- Formularze  
- Proste stylowanie w CSS  
- Małe przykłady układów stron  

To nie jest projekt komercyjny — odzwierciedla mój proces nauki.
